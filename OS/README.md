# OS
### 운영체제
#### 1.1 개요
---
###### 운영체제: 응용 프로그램의 실행을 제어하고 컴퓨터 시스템의 입출력 장치와 사용자 사이의 인터페이스 역할을 하는 프로그램
###### 운영체제 위치: 하드웨어와 사용자 혹은 응용 프로그램 사이
###### 운영체제 목적: 컴퓨터 시스템 자원의 효율적 관리와 시용자 편의성 제공
###### 운영체제 구성 요소:  CPU 관리 루틴, 메모리 관리 루틴, 파일 관리 루틴, 입출력 관리 루틴, 사용자 인터페이스 루틴
